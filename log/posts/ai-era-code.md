# AI 시대, 코드를 연주하다

새해 첫날에는 팀과 함께 "바이브 코딩 세션"을 열었다. 프롬프트 엔지니어링과 IDE 오토메이션을 엮어 악기처럼 연주하니 커서 이동 소리마저 박자를 맞췄다. 텍스트가 스크롤될 때마다 DAW의 파형 같은 감각이 들었고, 라이브러리 캐시를 비우는 소리조차 하이햇처럼 느껴졌다.

![라이브 코딩 세션의 감각을 기록한 사진](/log/images/ai-era.jpg)

세션의 핵심은 피드백 루프였다. 모델이 제안하는 코드 위에 내 감각을 얹으면 즉시 품질 측정이 돌아가고, 설계가 모호해질 때마다 3줄 요약을 강제해 사고의 겹을 줄였다. 커밋 훅에는 실험적인 린터를 걸어 두어, 즉흥 연주 와중에도 "핵심 리프"만 남도록 했다.

정리된 규칙은 플레이북에 바로 반영했다. 템포가 느려지는 순간에는 AI에게 "박자 유지"를 요청하고, 대규모 브랜치 병합 전에는 15분짜리 재밍 세션을 예약해 집중력을 다시 끌어올린다. 오후에는 이 리듬 그대로 빠르게 프로토타입 하나를 완성해 팀 채널에 올렸고, 동료들이 댓글로 솔로 파트를 이어가며 토론을 계속했다.

올해 목표는 이 바이브를 일상 업무에까지 스며들게 하는 것이다. 매일 아침 20분짜리 소규모 세션을 열어 프롬프트, 테스트, 배포 템포를 다시 맞추고 있다. AI는 악기가 아니라 밴드 동료라는 인식이 자리 잡을 때까지, 리허설은 계속될 예정이다.
